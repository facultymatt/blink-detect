Blink detect using getUserMedia and Canvas

The initial demo (Firefox only) works but also makes a lot of false positives and negatives if there's too much background movement.

Movement is a refinement of the initial demo to work in other browsers that support getUserMedia.

To limit the background interference we tried using a small fixed area, asking the user to keep their head still.

We then tried using a headtracker script to ignore the area outside of the user's face.
